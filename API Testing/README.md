Part 2: API Testing
URL To test
https://jsonplaceholder.typicode.com
Tasks
Using Postman (or any API client), create and execute test cases for the endpoints below.
For each test, include in your submission:
● Request details (URL, method, headers, body if any)
● Response details (status code, headers, body)
● Your short analysis of the result
● Include at least 1 edge case for each endpoint
Endpoints to Test:
1. GET /users – Retrieve all users.
2. GET /users/1 – Retrieve a specific user.
3. POST /posts – Create a new post with title and body.
4. PUT /posts/1 – Update an existing post.

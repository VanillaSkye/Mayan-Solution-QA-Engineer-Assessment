# Mayan-Solution-QA-Engineer-Assesment

Objective
Demonstrate your ability to perform manual testing, automation, and API testing on sample
applications to verify that core features work as expected.
Instructions
1. Review the user story and acceptance criteria.
2. Create and document test cases.
3. Execute manual testing.
4. Report any bugs or issues found.
5. Automate the main web application test flow using Playwright.
6. Perform API testing using the public JSONPlaceholder API.
7. Submit your work in a GitHub repository
